# ObsidianQ — Quantum Hybrid Technology

Ultra-modern futuristic single-page website for a Quantum Hybrid Technology company,
inspired by anti-gravity architecture and next-generation hybrid quantum-classical
optimization systems.

> **"Quantum Hybrid Optimization for Real-World Industrial Intelligence"**

## Features

- **Cinematic hero** — Anti-gravity quantum tower built procedurally in Three.js with
  energy core, levitating platforms, holographic data streams, and ambient particle field.
- **Live telemetry panel** — Animated qubit grid + glassmorphism solver KPIs.
- **Quantum infrastructure** — Interactive 8³ qubit lattice with real-time activity.
- **Logistics, industrial, solver, pipeline, sustainability, research** sections — all
  fully designed with glassmorphism cards, gradient accents, and reveal-on-scroll animation.
- **Holographic navbar** — Floating pill-shaped nav with backdrop blur and scroll state.
- **Hybrid solver showcase** — 3D constraint graph + KPI grid.
- **Sustainability bars** — Animated metrics about refrigerator-level energy use.
- **Research code card** — Ocean-inspired Python SDK preview.
- **Brand identity wall** — Concept names (ObsidianQ, Aetherion, Hyperion Hybrid, …).

## Tech

Pure HTML / CSS / JavaScript, no build step.

- Three.js (CDN) for the hero tower, qubit lattice, and constraint graph
- Custom CSS — dark futuristic theme, glassmorphism, gradient text, neon accents
- Vanilla JS — IntersectionObserver reveal, animated counters, mobile menu

## Run

Just open `index.html` in any modern browser, or serve the directory:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Structure

```
index.html
assets/

  js/quantum-tower.js   # Hero Three.js scene
  js/main.js            # Interactivity + secondary 3D scenes
```
